# DreamPlay

DreamPlay is Lionel Yu's project focused on narrow-key digital pianos for pianists with smaller hands.

## Current status

DreamPlay is in pre-production and prototype stage.
There are about 55 preorders and roughly $18K in preorder revenue reserved for prototype and manufacturing work, not normal operations.

## Current product priority

A current priority is the preorder buyer portal so customers can see:
- status
- amount paid
- selected choice
- Keep / Upgrade / Cancel actions

## Recent implementation work

Recent work includes:
- importing preorder and order data into `dreamplay-website-3`
- testing the `/my-reservation` login and buyer-access flow
- debugging Vercel deploy issues
