# Belgium Concert 2026 — Next Step-by-Step Execution Plan

## Goal
Move immediately from zero shipped marketing to a live, trackable campaign that can sell the first 100 tickets allocated to Lionel's channel, while preserving the Luc relationship and creating momentum for broader hall sales.

## 1. Decide the sales funnel this week
- Choose between:
  - a simple landing page with email capture and purchase CTA, or
  - direct-to-venue / Ticketmatic link
- Recommended default: use a simple landing page only if it can be shipped immediately without slowing the announcement.
- If there is any risk of delay, use direct-to-venue now and add a landing page later.

## 2. Gather assets and source-of-truth inputs
Collect the minimum inputs needed before copy is finalized:
- approved photos
- logo
- venue hero image
- current subscriber / follower counts for any claims
- confirmation on whether this is Lionel's only European date in 2026
- any Lions Club / Luc copy constraints
- official ticket links for standard and VIP paths

## 3. Lock the core message set
Draft one consistent messaging package that can be adapted per channel:
- master positioning paragraph
- short hook / headline
- CTA line
- Lionel-channel version
- Belgian-facing version that preserves the charity framing
- urgency version if “only European date” is verified true

## 4. Build the week-1 content calendar
Create the first week's execution calendar for approval:
- announcement video
- YouTube community post
- first email to list
- 3 to 5 short-form social posts
- one FAQ-style post covering date, location, and ticketing
- one urgency / countdown post

## 5. Draft the first outbound assets
Prepare first-pass drafts for:
- concert announcement email
- YouTube / community announcement copy
- short-form caption bank
- landing page copy or direct-link CTA copy
- Luc weekly update email draft

## 6. Set up tracking and reporting
Before launch, define a simple operating dashboard:
- ticket sales visibility against the 100-ticket goal
- weekly projection snapshot
- channel-by-channel traffic or response notes if available
- recurring update format for Luc

## 7. Launch wave one
Publish and send the first campaign wave:
- announcement post(s)
- first email
- first short-form content
- Luc update begins immediately

## 8. Review performance and adjust
After the first signals come in:
- review clicks, replies, geo interest, and sales signal
- adjust messaging and cadence
- decide whether to increase posting frequency
- decide when to push beyond the first 100 tickets if momentum is strong

## Main operating principle
Speed matters more than elegance right now. Do not let a nicer funnel or more polished asset pack delay the first live announcement.
